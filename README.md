# N2O for .NET

## Build & Run
Windows, Unix, Linux, Mac:

```cmd
..\n2o>dotnet run
```

## Notes
* No 3rd-party libs, just base default library
* Minimalism. Size of everything matters
* https://github.com/atemerev/skynet
* Make future for each connection to emulate CML lib of SML
* Avoid exceptions
